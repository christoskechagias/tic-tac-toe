## Run
### <b> 1. Clone repo </b> <br/>
<pre>
$ git clone https://github.com/kechayias/tic-tac-toe.git
$ cd tic-tac-toe
</pre>
  
### <b> 2. Run App </b> <br/>
<pre>
$ npm install
$ expo start -c
</pre>
